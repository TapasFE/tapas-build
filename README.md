# Tapas-build

## A command line tools for building React project.

------

### Install

``` shell
$ npm install tapas-build -g
```

### Usage

``` shell
$ tapas-build --index pathToYourIndex.html <entry> <output>
```

### Options

```
-h, --help
-p, --production
-i. --index
```
